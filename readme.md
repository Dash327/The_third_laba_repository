
        КРЕСТИКИ-НОЛИКИ для 2-ч игроков 


Описание:
Простая игра «Крестики-нолики» для двух человек с графическим интерфейсом.
Поддерживается настройка размера поля (от 3×3 до 6×6).

Требования:
- Python 3.6 или новее
- Библиотека tkinter

Как запустить:
1. Убедитесь, что установлен Python 
2. Сохраните код игры в файл, например: `game.py`
3. Откройте терминал / командную строку.
4. Перейдите в папку с файлом:
      cd путь/к/папке
5. Запустите игру командой:
      python game.py

Управление:
- Левый клик мышью по клетке — сделать ход (X или O поочерёдно).
- Меню «Игра»:
    • «Новая игра» — начать заново;
    • «Размер поля...» — изменить размер доски (только до начала игры);
    • «Выход» — закрыть приложение.
